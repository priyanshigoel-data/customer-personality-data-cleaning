# Customer Personality Analysis - Data Cleaning

## 📌 Objective
Clean and preprocess the Customer Personality Analysis dataset using Python and Pandas.

## 🛠️ Tools Used
- Python
- Pandas
- Jupyter Notebook

## Skills Demonstrated
- Data Cleaning
- Data Preprocessing
- Missing Value Handling
- Data Validation
- Data Transformation
- 
## 📂 Dataset
Customer Personality Analysis (Marketing Campaign Dataset)

## ✅ Data Cleaning Steps
- Loaded the dataset
- Explored dataset structure
- Checked missing values
- Filled missing Income values using the median
- Renamed column names for consistency
- Converted Dt_Customer to datetime format
- Removed duplicate records
- Dropped constant columns (Z_CostContact and Z_Revenue)
- Validated the cleaned dataset
- Exported the cleaned dataset
